# 🎵 Music Player App

A responsive music player built using **HTML, SCSS, and Vanilla JavaScript**.

## 🚀 Live Demo

https://houda111111.github.io/Music-Player-App/

## ✨ Features

* Play / Pause songs
* Next / Previous controls
* Dynamic song switching
* Progress bar updates while music plays
* Click on progress bar to seek through audio
* Display current time and duration
* Responsive UI design
* Dynamic data rendering using JavaScript arrays

## 🛠️ Technologies Used

* HTML5
* SCSS
* JavaScript (Vanilla JS)
* Audio API
